# Java Bank 

A simple console-based banking app made in Java.

This was a beginner project to practice Object-Oriented Programming concepts like classes, objects, and methods.

## What it does

- Select from 3 bank accounts
- Login with a PIN (3 attempts allowed)
- Check balance, deposit, and withdraw money
- View mini statement and change PIN

## How to run

javac Main.java BankAccount.java
java Main


## What I learned

- Creating and using classes in Java
- Taking user input with `Scanner`
- Using arrays and loops
- Basic input validation with try-catch
